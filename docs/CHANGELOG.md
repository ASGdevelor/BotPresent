# Журнал изменений

## 1.1.0 — 2026-07-10

### Изменено

- Кнопка **«Провести исследование»** заменена на **«Лидогенерация»**.
- Одношаговый запрос заменён типизированной анкетой из пяти вопросов.
- README и пользовательская документация обновлены под новый сценарий.
- Описание пакета обновлено, версия повышена до `1.1.0`.

### Добавлено

- Поиск сайтов через публичную HTML-выдачу Bing или анализ явно заданных URL.
- Извлечение названия, описания, контактных страниц и публичных Telegram handles.
- Markdown-отчёт со списком компаний и контактами.
- Типы `LeadCriteria`, `CompanyLead`, `TelegramContact` и `LeadGenerationResult`.
- Частичная устойчивость: недоступность одного сайта не отменяет анализ остальных.
- Проверки HTTP-статусов, типа и размера ответа, DNS, редиректов и тайм-аутов.
- Unit-тесты парсинга лидов и сетевой защиты.

### Безопасность

- Запрещены запросы к localhost, приватным и служебным IP-адресам.
- Не сохраняются Telegram invite-ссылки и аккаунты ботов.
- Собираются только контакты, опубликованные на открытых страницах.

### Удалено

- Интеграция с Wikipedia и файл `src/services/research.ts`.

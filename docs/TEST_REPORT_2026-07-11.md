## PDF и ошибки

- PDF печатается из заполненного `index.html`, а не из отдельного упрощённого шаблона.
- Формат печати — 16:9; один `<section>` равен одной PDF-странице.
- При недоступном CDN работает встроенный canvas-fallback графиков.
- Chrome/Edge ограничен тайм-аутом; на Windows зависшее дерево процессов принудительно завершается.
- Временный профиль браузера создаётся в системном temp, удаляется после штатного выхода и больше не засоряет каталог презентации.
- Если браузер отсутствует или запрещён политикой ОС, HTML всё равно возвращается пользователю, а бот явно сообщает, почему PDF не создан.

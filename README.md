# BotPresent

Telegram-бот на Bun, TypeScript и [grammY](https://grammy.dev/) с двумя основными действиями:

- **📊 Создать презентацию** — запрашивает тему и отправляет готовый файл PowerPoint `.pptx` из шести слайдов.
- **🔎 Провести исследование** — ищет связанные материалы в русскоязычной Wikipedia и отправляет отчёт `.md` со ссылками на источники.

Проект работает без платных API и ключей AI. Для запуска нужен только токен Telegram-бота.

## Быстрый старт

### 1. Создайте Telegram-бота

1. Откройте [@BotFather](https://t.me/BotFather).
2. Выполните `/newbot` и следуйте инструкциям.
3. Сохраните полученный токен.

### 2. Настройте проект

Установите [Bun](https://bun.sh/), затем выполните:

```bash
git clone https://github.com/ASGdevelor/BotPresent.git
cd BotPresent
bun install
```

Скопируйте `.env.example` в `.env` и укажите токен:

```env
BOT_TOKEN=ваш_токен_от_BotFather
```

Для совместимости также распознаётся переменная `TOKEN`, но в новых конфигурациях рекомендуется `BOT_TOKEN`.

### 3. Запустите

```bash
bun run start
```

Режим разработки с автоматическим перезапуском:

```bash
bun run dev
```

## Команды

| Команда | Назначение |
| --- | --- |
| `/start` | Запустить бота и показать две кнопки |
| `/menu` | Повторно показать главное меню |
| `/cancel` | Отменить ожидание темы |

## Как работает бот

Состояние выбранного действия хранится в памяти процесса до получения следующего текстового сообщения. После генерации временный `.pptx` удаляется. Отчёт исследования создаётся в памяти и не сохраняется на диске.

Wikipedia — стартовая точка, а не первичный источник. Для важных решений проверяйте дату, авторство и исходные публикации. Текущая презентация представляет собой редактируемый базовый шаблон; фактическое наполнение следует адаптировать под задачу и аудиторию.
